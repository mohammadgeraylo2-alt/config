# 🤖 VLESS Bot

ربات تلگرامی که کانفیگ‌های VLESS رایگان رو تست می‌کنه و می‌فرسته.

## راه‌اندازی روی Railway

1. این ریپو رو Fork کن
2. توی Railway یه پروژه جدید بساز و به GitHub وصل کن
3. متغیر محیطی زیر رو اضافه کن:
   - `BOT_TOKEN` = توکن ربات تلگرامت

## متغیرهای محیطی

| متغیر | توضیح |
|-------|--------|
| `BOT_TOKEN` | توکن دریافتی از @BotFather |
